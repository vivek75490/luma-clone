We are seeking a skilled Data Infrastructure Engineer to join our team and help build robust, scalable data systems that power our AI platform. This role will focus on designing and implementing data pipelines, storage solutions, and infrastructure that can handle massive scale and complex AI workloads.

Responsibilities

  • Design and implement scalable data infrastructure and pipelines
  • Build and optimize data storage solutions for AI workloads
  • Develop data processing frameworks and ETL systems
  • Implement data governance, quality, and security measures
  • Create and maintain data APIs and access patterns
  • Optimize data performance and query optimization
  • Implement monitoring and observability for data systems
  • Collaborate with AI/ML teams to optimize data workflows
  • Participate in data architecture and infrastructure planning

Background

  • 4+ years of experience in data infrastructure or related field
  • Strong proficiency in Python, Java, Scala, or similar languages
  • Experience with big data technologies (Spark, Hadoop, Kafka, etc.)
  • Knowledge of data warehouses and data lakes (Snowflake, BigQuery, etc.)
  • Experience with cloud data platforms (AWS, GCP, Azure)
  • Understanding of data modeling, ETL processes, and data governance
  • Experience with real-time data processing and streaming
  • Knowledge of data security, privacy, and compliance
  • Bachelor's degree in Computer Science, Engineering, or related field
