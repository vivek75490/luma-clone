Software Engineer - Test Automation & QA
LocationPalo Alto, CA
Employment TypeFull time
Location TypeHybrid
DepartmentProduct Engineering
Compensation$180K - $240K

We are seeking a Test Automation & QA Engineer to join our team and help ensure the quality and reliability of our AI platform. This role will focus on building comprehensive test automation frameworks, implementing quality assurance processes, and working closely with development teams to maintain high software quality standards.

Responsibilities

  • Design and implement automated testing frameworks for web and mobile applications
  • Develop and maintain test suites for API testing, integration testing, and end-to-end testing
  • Create and execute test plans for new features and regression testing
  • Work with development teams to integrate testing into CI/CD pipelines
  • Perform manual testing when necessary and document test cases
  • Analyze test results and report bugs with detailed reproduction steps
  • Collaborate with product managers to understand requirements and acceptance criteria
  • Implement performance testing and load testing strategies
  • Mentor junior QA engineers and promote testing best practices

Background

  • 4+ years of experience in software testing and quality assurance
  • Strong programming skills in Python, JavaScript, or similar languages
  • Experience with test automation frameworks (Selenium, Playwright, Cypress, etc.)
  • Knowledge of API testing tools and methodologies
  • Experience with CI/CD tools and integration (Jenkins, GitHub Actions, etc.)
  • Understanding of software development lifecycle and agile methodologies
  • Experience with performance testing tools (JMeter, K6, etc.)
  • Strong analytical and problem-solving skills
  • Bachelor's degree in Computer Science, Engineering, or related field

What We Offer

  • Competitive salary with equity options
  • Comprehensive health, dental, and vision insurance
  • 401(k) matching and flexible spending accounts
  • Flexible work arrangements and remote work options
  • Professional development and conference attendance
  • Collaborative and innovative work environment
  • Opportunity to shape testing practices for AI products
  • Access to cutting-edge testing tools and technologies

Note: This role requires strong technical skills combined with attention to detail and quality mindset
