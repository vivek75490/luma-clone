LocationRemote
Employment TypeFull time
Location TypeRemote

We are seeking a Senior Machine Learning Engineer to join our research team and help develop hardware abstractions and performance optimization techniques for our large-scale AI models. This role will focus on creating efficient, portable, and scalable solutions that can run optimally across different hardware platforms and architectures.

Responsibilities

  • Design and implement hardware abstraction layers for ML frameworks
  • Develop performance optimization techniques for various hardware platforms
  • Create efficient kernels and operators for GPU, TPU, and CPU execution
  • Optimize memory usage and data movement patterns
  • Implement quantization and compression techniques for model deployment
  • Develop tools for performance profiling and analysis
  • Create portable ML code that can run efficiently across different architectures
  • Collaborate with hardware vendors to optimize for new platforms
  • Mentor junior engineers and contribute to technical strategy

Background

  • 7+ years of experience in machine learning engineering or systems programming
  • Deep understanding of computer architecture and hardware optimization
  • Strong programming skills in C++, CUDA, and Python
  • Experience with ML frameworks (PyTorch, TensorFlow, JAX) internals
  • Knowledge of GPU programming and parallel computing
  • Experience with performance optimization and profiling tools
  • Understanding of numerical computing and linear algebra
  • Experience with distributed systems and high-performance computing
  • Master's degree in Computer Science, Engineering, or related field (PhD preferred)

  • 401(k) matching and flexible spending accounts
  • Flexible work arrangements and remote work options
  • Professional development and conference attendance
  • Collaborative and innovative work environment
  • Opportunity to work on cutting-edge hardware optimization
  • Access to state-of-the-art compute infrastructure
